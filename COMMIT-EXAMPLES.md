# Git 커밋 예시 파일

이 파일은 Git 커밋 연습용으로 생성되었습니다.

## 작성일
2025-01-20

## 목적
- Git 커밋 컨벤션 학습
- 실제 커밋 워크플로우 실습

## 커밋 컨벤션 요약

### 형식
```
[태그] 제목 (#이슈번호)

본문 (선택)
- 변경사항 1
- 변경사항 2
```

### 태그 종류
- `Feat`: 기능 추가
- `Fix`: 버그 수정
- `Refactor`: 리팩토링
- `Docs`: 문서 작성
- `Test`: 테스트 추가
- `Style`: 코드 포맷팅
- `Chore`: 설정/빌드

## 예시 커밋 메시지

### 간단한 커밋
```bash
[Docs] 커밋 예시 파일 추가 (#001)
```

### 상세한 커밋 (권장)
```bash
[Docs] 커밋 예시 파일 추가 (#001)

- Git 커밋 컨벤션 요약 추가
- 태그 종류 및 예시 정리
- 실습용 파일 생성
```
